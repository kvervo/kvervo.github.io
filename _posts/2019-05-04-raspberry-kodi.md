Steps to install

- Get Raspberry PI
- Install Raspebian
- First boot (without keyboard and mouse)
  - Configure Pi 
    - Flash SD
    - Add ssh file (MacOS: touch ssh in root | Windows: type NUL >> ssh)
- Install Kodi
- Configure Kodi
  - Advanced settings.. since standard guisettings do not work.
